# Traffic Light Controller using Verilog

## Overview
This project implements a finite state machine (FSM) based traffic light controller using Verilog HDL.

## Features
- Moore FSM design
- Sequential state transitions
- RED, GREEN, YELLOW states
- Testbench included

## Folder Structure

rtl/
- RTL source code

tb/
- Testbench files

docs/
- Simulation waveform images

## FSM Flow

RED → GREEN → YELLOW → RED

## Tools Used
- Verilog HDL
- ModelSim / Vivado
